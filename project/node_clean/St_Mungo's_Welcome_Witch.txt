<h2><span class="mw-headline" id="Biography">Biography</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/wiki/St_Mungo%27s_Welcome_Witch?veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Edit section: Biography">edit</a><span class="mw-editsection-divider"> | </span><a href="/wiki/St_Mungo%27s_Welcome_Witch?action=edit&amp;section=1" title="Edit section: Biography"><svg class="wds-icon wds-icon-tiny section-edit-pencil-icon"><use xlink:href="#wds-icons-pencil-tiny"></use></svg>edit source</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>When <a href="/wiki/Harry_Potter" title="Harry Potter">Harry Potter</a> visited <a href="/wiki/Arthur_Weasley" title="Arthur Weasley">Arthur Weasley</a> with the <a href="/wiki/Weasley_family" title="Weasley family">Weasleys</a> at <a href="/wiki/Christmas" title="Christmas">Christmas</a> <a href="/wiki/1995" title="1995">1995</a>, the Welcome Witch appeared to be a typical secretary type; somewhat rude, brief, and monotonous, which was likely used to be humorous when <a href="/wiki/Harry_Potter" title="Harry Potter">Harry Potter</a> read a sign, reading "Our Welcome Witch will be happy to assist you".<sup id="cite_ref-OOTP22_1-9" class="reference"><a href="#cite_note-OOTP22-1">&#91;1&#93;</a></sup>
</p>
