<b>Madame Magee</b> was a <a href="/wiki/Muggle" class="mw-redirect" title="Muggle">No-Maj</a> entertainer in the early <a href="/wiki/20th_century" title="20th century">20th century</a>. In <a href="/wiki/1926" title="1926">1926</a>, posters were put up in an alleyway in <a href="/wiki/New_York" title="New York">New York</a> to promote a one-night-only performance by Madam Magee at the <a href="/wiki/Brobeck_Theatre" title="Brobeck Theatre">Brobeck Theatre</a>, which was to be held on Saturday <a href="/wiki/4_December" title="4 December">4 December</a>.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup>
