<b>Wilfred Elphick</b> (<a href="/wiki/1112" class="mw-redirect" title="1112">1112</a>-<a href="/wiki/1199" class="mw-redirect" title="1199">1199</a>) was the first <a href="/wiki/Wizardkind" title="Wizardkind">wizard</a> to be gored by an <a href="/wiki/Africa" title="Africa">African</a> <a href="/wiki/Erumpent" title="Erumpent">Erumpent</a>, which is an extremely unlikely event, given the fact that the <a href="/wiki/Erumpent_horn" title="Erumpent horn">horn</a> of an Erumpent almost always explodes on contact, due to the <a href="/wiki/Exploding_Fluid" title="Exploding Fluid">fluid</a> it contains. On his <a href="/wiki/Chocolate_Frog_Card" title="Chocolate Frog Card">Chocolate Frog Card</a>, he is shown in bandages, so it can be presumed that he survived this attack.
</p>
<h2><span class="mw-headline" id="Etymology">Etymology</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/wiki/Wilfred_Elphick?veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Edit section: Etymology">edit</a><span class="mw-editsection-divider"> | </span><a href="/wiki/Wilfred_Elphick?action=edit&amp;section=1" title="Edit section: Etymology"><svg class="wds-icon wds-icon-tiny section-edit-pencil-icon"><use xlink:href="#wds-icons-pencil-tiny"></use></svg>edit source</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Wilfred means "wanting peace," from the Old English <i>wil</i>, "desire," and <i>fred</i>, "peace."<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup> The surname Elphick derived from the Old English given name Aelfheah, which meant "high elf," from <i>aelf</i>, "elf," and <i>heah</i>, "high."<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup>
</p>
