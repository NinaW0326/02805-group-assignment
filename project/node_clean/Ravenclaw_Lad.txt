<h2><span class="mw-headline" id="Biography">Biography</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/wiki/Ravenclaw_Lad?veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Edit section: Biography">edit</a><span class="mw-editsection-divider"> | </span><a href="/wiki/Ravenclaw_Lad?action=edit&amp;section=1" title="Edit section: Biography"><svg class="wds-icon wds-icon-tiny section-edit-pencil-icon"><use xlink:href="#wds-icons-pencil-tiny"></use></svg>edit source</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>He lived somewhere in <a href="/wiki/Great_Britain" title="Great Britain">Great Britain</a> or <a href="/wiki/Ireland" title="Ireland">Ireland</a> and started attending Hogwarts on <a href="/wiki/1_September" title="1 September">1 September</a> <a href="/wiki/1984" title="1984">1984</a>.<sup id="cite_ref-Transfiguration_2-5" class="reference"><a href="#cite_note-Transfiguration-2">&#91;2&#93;</a></sup>
</p><p>He had <a href="/wiki/Transfiguration_(class)" title="Transfiguration (class)">Transfiguration</a> class with <a href="/wiki/Jacob%27s_sibling" title="Jacob&#39;s sibling">Jacob's sibling</a> and didn't like the other students talking during lectures.<sup id="cite_ref-Transfiguration_2-6" class="reference"><a href="#cite_note-Transfiguration-2">&#91;2&#93;</a></sup>
</p><p>By his <a href="/wiki/Fifth_year" title="Fifth year">fifth year</a>, this student attended <a href="/wiki/Minerva_McGonagall" title="Minerva McGonagall">Professor McGonagall</a>'s lectures on <a href="/wiki/Transfiguration" title="Transfiguration">transforming</a> <a href="/wiki/Ferret" title="Ferret">ferrets</a> to feather dusters<sup id="cite_ref-Transfiguration_2-7" class="reference"><a href="#cite_note-Transfiguration-2">&#91;2&#93;</a></sup> and <a href="/wiki/Switching_Spell" title="Switching Spell">switching</a> ferrets with feather dusters.<sup id="cite_ref-HM5ch5_3-0" class="reference"><a href="#cite_note-HM5ch5-3">&#91;3&#93;</a></sup>
</p>
