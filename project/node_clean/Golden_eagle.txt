<p>The <b>golden eagle</b> (<i>Aquila chrysaetos</i>) is a species of <a href="/wiki/Eagle" title="Eagle">eagle</a> endemic to much of the <a href="http://en.wikipedia.org/wiki/Northern_Hemisphere" class="extiw" title="wikipedia:Northern Hemisphere">Northern Hemisphere</a>, including parts of <a href="/wiki/Europe" title="Europe">Europe</a>, <a href="/wiki/Asia" title="Asia">Asia</a>, and <a href="/wiki/North_America" title="North America">North America</a>.<sup id="cite_ref-wiki1_1-0" class="reference"><a href="#cite_note-wiki1-1">&#91;1&#93;</a></sup>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/wiki/Golden_eagle?veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Edit section: History">edit</a><span class="mw-editsection-divider"> | </span><a href="/wiki/Golden_eagle?action=edit&amp;section=1" title="Edit section: History"><svg class="wds-icon wds-icon-tiny section-edit-pencil-icon"><use xlink:href="#wds-icons-pencil-tiny"></use></svg>edit source</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><i><a href="/wiki/Transfiguration_Today" title="Transfiguration Today">Transfiguration Today</a></i> ran a story in a <a href="/wiki/1926" title="1926">1926</a> issue about how a <a href="/wiki/Wizardkind" title="Wizardkind">witch</a> had "magic[ked] herself" into a golden eagle (presumably via an <a href="/wiki/Animagus" title="Animagus">Animagus</a> <a href="/wiki/Transformation" title="Transformation">transformation</a>).<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup>
</p><p>While investigating a <a href="/wiki/Magic" title="Magic">magical</a> <a href="/wiki/Creature" title="Creature">creature</a> attack, <a href="/wiki/Mathilda_Grimblehawk" title="Mathilda Grimblehawk">Mathilda Grimblehawk</a> and <a href="/wiki/Mathilda_Grimblehawk%27s_partner" title="Mathilda Grimblehawk&#39;s partner">her partner</a> discovered a <a href="/wiki/Feather" title="Feather">feather</a> in <a href="/wiki/Percival_Shacklehorn" title="Percival Shacklehorn">Percival Shacklehorn</a>'s <a href="/wiki/Percival_Shacklehorn%27s_tent" title="Percival Shacklehorn&#39;s tent">tent</a>, taking it to be examined by <a href="/wiki/Sage_Bragnam" title="Sage Bragnam">Sage Bragnam</a>.<sup id="cite_ref-fb-cases_3-0" class="reference"><a href="#cite_note-fb-cases-3">&#91;3&#93;</a></sup> Sage told them that it did not come from a <a href="/wiki/Magic" title="Magic">magical</a> <a href="/wiki/Beast" title="Beast">beast</a>, but rather a non-magical golden eagle, which he revealed nested year-round in mountainous regions.<sup id="cite_ref-fb-cases_3-1" class="reference"><a href="#cite_note-fb-cases-3">&#91;3&#93;</a></sup>
</p><p>The <a href="/wiki/Eagle" title="Eagle">eagle</a> mascot of <a href="/wiki/Ravenclaw" title="Ravenclaw">Ravenclaw</a> is sometimes depicted as a golden eagle.<sup id="cite_ref-ravenclaw_4-0" class="reference"><a href="#cite_note-ravenclaw-4">&#91;4&#93;</a></sup>
</p>
