<b>Maud Montague</b> was a <a href="/wiki/Wizardkind" title="Wizardkind">witch</a> whose <a href="/wiki/Portrait" title="Portrait">portrait</a> hung at <a href="/wiki/Hogwarts_Castle" title="Hogwarts Castle">Hogwarts Castle</a> in the <a href="/wiki/1990s" title="1990s">1990s</a>.<sup id="cite_ref-HPLE_2-0" class="reference"><a href="#cite_note-HPLE-2">&#91;2&#93;</a></sup>
</p><p>It is possible that Maud is a forerunner of the <a href="/wiki/Montague_family" title="Montague family">Montague family</a>, which was active in the <a href="/wiki/20th_century" title="20th century">20th century</a>.
</p>
