<p><b>Ingolfr the Iambic</b> was a <a href="/wiki/Norway" title="Norway">Norwegian</a> <a href="/wiki/Wizardkind" title="Wizardkind">wizard</a> and poet in the early <a href="/wiki/1400s" title="1400s">1400s</a>. He wrote a <a href="/wiki/Seeker%27s_verse" title="Seeker&#39;s verse">verse</a> about the game of <a href="/wiki/Quidditch" title="Quidditch">Quidditch</a> that showed how the game had spread to <a href="/wiki/Europe" title="Europe">Europe</a>.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup>
</p>
<h2><span class="mw-headline" id="Etymology">Etymology</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/wiki/Ingolfr_the_Iambic?veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Edit section: Etymology">edit</a><span class="mw-editsection-divider"> | </span><a href="/wiki/Ingolfr_the_Iambic?action=edit&amp;section=1" title="Edit section: Etymology"><svg class="wds-icon wds-icon-tiny section-edit-pencil-icon"><use xlink:href="#wds-icons-pencil-tiny"></use></svg>edit source</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul><li>The name Ingólfr, was derived from the name of the Germanic god <i>Ing</i> combined with <i>úlfr</i> meaning "wolf".</li>
<li>Iambic is likely a reference to <a href="http://en.wikipedia.org/wiki/Iambic_pentameter" class="extiw" title="wikipedia:Iambic pentameter">iambic pentameter</a>, a common poetry metre best known for its extensive use by <a href="/wiki/William_Shakespeare" title="William Shakespeare">William Shakespeare</a>. Oddly, his only known poem is not in an iambic metre, but instead anapestic tetrameter.</li></ul>
