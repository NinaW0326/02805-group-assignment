<a href="/wiki/Tarquin_McTavish" title="Tarquin McTavish">Tarquin McTavish</a> had a <a href="/wiki/Non-magic_people" title="Non-magic people">Muggle</a> <b>neighbour</b>. For unknown reasons, Tarquin <a href="/wiki/Magic" title="Magic">magically</a> trapped this neighbour inside a <a href="/wiki/Tarquin_McTavish%27s_teakettle" title="Tarquin McTavish&#39;s teakettle">teakettle</a>, which led to the <a href="/wiki/Wizardkind" title="Wizardkind">wizard</a>'s imprisonment.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup>

<h2><span class="mw-headline" id="Etymology">Etymology</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/wiki/Tarquin_McTavish%27s_neighbour?veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Edit section: Etymology">edit</a><span class="mw-editsection-divider"> | </span><a href="/wiki/Tarquin_McTavish%27s_neighbour?action=edit&amp;section=1" title="Edit section: Etymology"><svg class="wds-icon wds-icon-tiny section-edit-pencil-icon"><use xlink:href="#wds-icons-pencil-tiny"></use></svg>edit source</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The name "Tarquin" comes from the Roman name Tarquinius which is of unknown meaning, possibly Etruscan in origin. This was the name of two early kings of Rome.
