This <b>unnamed <a href="/wiki/Ireland" title="Ireland">Irish</a> man</b> met <a href="/wiki/Rubeus_Hagrid" title="Rubeus Hagrid">Rubeus Hagrid</a> in the <a href="/wiki/Leaky_Cauldron" title="Leaky Cauldron">Leaky Cauldron</a> in <a href="/wiki/1990" title="1990">1990</a>. He sold Hagrid the <a href="/wiki/Fluffy" title="Fluffy">three-headed dog</a> he owned. Hagrid named this dog <a href="/wiki/Fluffy" title="Fluffy">Fluffy</a>, and, the <a href="/wiki/1991" title="1991">following year</a>, lent him to <a href="/wiki/Albus_Dumbledore" title="Albus Dumbledore">Albus Dumbledore</a>, who used him to guard the <a href="/wiki/Philosopher%27s_Stone" title="Philosopher&#39;s Stone">Philosopher's Stone</a>.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup>
</p><p><br />

<h2><span class="mw-headline" id="Behind_the_scenes">Behind the scenes</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/wiki/Unidentified_Irishman?veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Edit section: Behind the scenes">edit</a><span class="mw-editsection-divider"> | </span><a href="/wiki/Unidentified_Irishman?action=edit&amp;section=1" title="Edit section: Behind the scenes"><svg class="wds-icon wds-icon-tiny section-edit-pencil-icon"><use xlink:href="#wds-icons-pencil-tiny"></use></svg>edit source</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul><li>In the novel <i><a href="/wiki/Harry_Potter_and_the_Philosopher%27s_Stone" title="Harry Potter and the Philosopher&#39;s Stone">Harry Potter and the Philosopher's Stone</a></i>, <a href="/wiki/Rubeus_Hagrid" title="Rubeus Hagrid">Hagrid</a> states that he bought <a href="/wiki/Fluffy" title="Fluffy">Fluffy</a> from "a <a href="/wiki/Greece" title="Greece">Greek</a> <a href="/wiki/Greek_man" title="Greek man">man</a>", rather than "an Irish feller,". Because of this change in the film, it loses <a href="/wiki/J._K._Rowling" title="J. K. Rowling">J. K. Rowling</a>'s reference to <a href="http://en.wikipedia.org/wiki/Greek_mythology" class="extiw" title="wikipedia:Greek mythology">Greek mythology</a>, in which there was a three-head dog, <a href="http://en.wikipedia.org/wiki/Cerberus" class="extiw" title="wikipedia:Cerberus">Cerberus</a>, that guarded the gates of <a href="http://en.wikipedia.org/wiki/Hades" class="extiw" title="wikipedia:Hades">Hades</a>. However, <a href="/wiki/Steve_Kloves" title="Steve Kloves">Steve Kloves</a>, the film's screenwriter, stated that he originally used "Greek chappie," but did not know how the line got changed.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup></li></ul>
