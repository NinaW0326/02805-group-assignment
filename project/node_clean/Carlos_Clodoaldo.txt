<b>Carlos Clodoaldo</b><sup id="cite_ref-FN1_1-0" class="reference"><a href="#cite_note-FN1-1">&#91;1&#93;</a></sup> was a <a href="/wiki/Brazil" title="Brazil">Brazilian</a> <a href="/wiki/Wizardkind" title="Wizardkind">wizard</a> and a <a href="/wiki/Beater" title="Beater">Beater</a> for the <a href="/wiki/2014" title="2014">2014</a> <a href="/wiki/Brazilian_National_Quidditch_team" title="Brazilian National Quidditch team">Brazilian National Quidditch team</a>. According to <a href="/wiki/Ludovic_Bagman" title="Ludovic Bagman">Ludovic Bagman</a> in a report to the <i><a href="/wiki/Daily_Prophet" title="Daily Prophet">Daily Prophet</a></i>, American Beaters <a href="/wiki/Lucas_Picquery" title="Lucas Picquery">Picquery</a> and <a href="/wiki/Pringle" title="Pringle">Pringle</a> would have to "<i>find better form</i>" if they were to best the Brazilians <a href="/wiki/Rafael_Santos" title="Rafael Santos">Santos</a> and Clodoaldo in their semi-final match on <a href="/wiki/4_July" title="4 July">4 July</a>, <a href="/wiki/2014" title="2014">2014</a>.<sup id="cite_ref-PM_2-0" class="reference"><a href="#cite_note-PM-2">&#91;2&#93;</a></sup>
