<h2><span class="mw-headline" id="Biography">Biography</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/wiki/Unidentified_male_bald_Death_Eater?veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Edit section: Biography">edit</a><span class="mw-editsection-divider"> | </span><a href="/wiki/Unidentified_male_bald_Death_Eater?action=edit&amp;section=1" title="Edit section: Biography"><svg class="wds-icon wds-icon-tiny section-edit-pencil-icon"><use xlink:href="#wds-icons-pencil-tiny"></use></svg>edit source</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>This Death Eater, along with <a href="/wiki/Unidentified_Male_Death_Eater_(VIII)" title="Unidentified Male Death Eater (VIII)">another Death Eater</a>, <a href="/wiki/Raid_on_the_Hogwarts_Express" title="Raid on the Hogwarts Express">raided</a> the <a href="/wiki/Hogwarts_Express" title="Hogwarts Express">Hogwarts Express</a> on <a href="/wiki/1_September" title="1 September">1 September</a>, <a href="/wiki/1997" title="1997">1997</a>, to search the train for <a href="/wiki/Harry_Potter" title="Harry Potter">Harry Potter</a>. They were both disappointed to find he was absent, with <a href="/wiki/Neville_Longbottom" title="Neville Longbottom">Neville Longbottom</a> insulting them as "losers".<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup>
</p><p>This Death Eater was at <a href="/wiki/Gringotts_Wizarding_Bank" title="Gringotts Wizarding Bank">Gringotts Wizarding Bank</a> as <a href="/wiki/Harry_Potter" title="Harry Potter">Harry Potter</a>, <a href="/wiki/Ronald_Weasley" title="Ronald Weasley">Ronald Weasley</a>, <a href="/wiki/Hermione_Granger" title="Hermione Granger">Hermione Granger</a> and <a href="/wiki/Griphook" title="Griphook">Griphook</a> staged a <a href="/wiki/Break-in_of_Gringotts_Wizarding_Bank_(1998)" title="Break-in of Gringotts Wizarding Bank (1998)">break-in</a> to retrieve <a href="/wiki/Helga_Hufflepuff%27s_Cup" class="mw-redirect" title="Helga Hufflepuff&#39;s Cup">Helga Hufflepuff's Cup</a> from the <a href="/wiki/Lestrange_Vault" title="Lestrange Vault">Lestrange Vault</a>.
</p><p>Later that day, he was <a href="/wiki/Killing_Curse" title="Killing Curse">killed</a>&#160;by <a href="/wiki/Tom_Riddle" title="Tom Riddle">Lord Voldemort</a> in a fit of rage at <a href="/wiki/Malfoy_Manor" title="Malfoy Manor">Malfoy Manor</a>, after the <a href="/wiki/Dark_Wizard" title="Dark Wizard">Dark Lord</a>&#160;was informed by <a href="/wiki/Goblin" title="Goblin">Goblins</a> working at Gringotts that <a href="/wiki/Harry_Potter" title="Harry Potter">Harry Potter</a> had managed to steal <a href="/wiki/Helga_Hufflepuff%27s_Cup" class="mw-redirect" title="Helga Hufflepuff&#39;s Cup">Helga Hufflepuff's Cup</a> from him.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup>
</p>
