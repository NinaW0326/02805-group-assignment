<b>Lara Von Sigg</b> was an <a href="/wiki/United_States_of_America" title="United States of America">American</a> <a href="/wiki/Wizardkind" title="Wizardkind">witch</a> and reporter for the <i><a href="/wiki/The_Wizard%27s_Voice" title="The Wizard&#39;s Voice">The Wizard's Voice</a></i> in <a href="/wiki/1926" title="1926">1926</a>.<sup id="cite_ref-newspaper_1-0" class="reference"><a href="#cite_note-newspaper-1">&#91;1&#93;</a></sup> Reporting from <a href="/wiki/Houston" title="Houston">Houston</a>, she wrote an article revealing the secret <a href="/wiki/Wand_core" title="Wand core">wand core</a> used by <a href="/wiki/New_Orleans" title="New Orleans">New Orleans</a> <a href="/wiki/Wandmaker" title="Wandmaker">wandmaker</a> <a href="/wiki/Violetta_Beauvais" title="Violetta Beauvais">Violetta Beauvais</a> to be <a href="/wiki/Rougarou" title="Rougarou">rougarou</a> <a href="/wiki/Rougarou_hair" title="Rougarou hair">hair</a>, which was published in the <a href="/wiki/6_December" title="6 December">6 December</a> 1926 issue of the newspaper.<sup id="cite_ref-newspaper_1-1" class="reference"><a href="#cite_note-newspaper-1">&#91;1&#93;</a></sup>
