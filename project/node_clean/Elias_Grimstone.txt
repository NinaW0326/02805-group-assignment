<b>Elias Grimstone</b> (fl. 1879) was a <a href="/wiki/Broom-maker" title="Broom-maker">broom-maker</a> from <a href="/wiki/Portsmouth" title="Portsmouth">Portsmouth</a>. In <a href="/wiki/1879" class="mw-redirect" title="1879">1879</a>, Elias created the <a href="/wiki/Oakshaft_79" title="Oakshaft 79">Oakshaft 79</a> <a href="/wiki/Broomstick" title="Broomstick">broomstick</a>.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup>

<h2><span class="mw-headline" id="Etymology">Etymology</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/wiki/Elias_Grimstone?veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Edit section: Etymology">edit</a><span class="mw-editsection-divider"> | </span><a href="/wiki/Elias_Grimstone?action=edit&amp;section=2" title="Edit section: Etymology"><svg class="wds-icon wds-icon-tiny section-edit-pencil-icon"><use xlink:href="#wds-icons-pencil-tiny"></use></svg>edit source</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b>Elias</b>&#160;is another version of the Hebrew name "Eliyahu", which is derived from 'ēlīyāhū, meaning "Jehovah is God".
</p><p>The name Grimstone&#160;is derived from the general case of the Old Norman personal&#160;"Grimr", and the Old English tun, which&#160;means settlement or town.
</p>
