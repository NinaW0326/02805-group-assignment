<b>Alec Webb</b> was a <a href="/wiki/Great_Britain" title="Great Britain">British</a> <a href="/wiki/Non-magic_people" title="Non-magic people">Muggle</a> concert director. He directed a performance of <a href="/wiki/Wolfgang_Amadeus_Mozart" title="Wolfgang Amadeus Mozart">Mozart</a>'s <i><a href="/wiki/Sinfonia_Concertanti" title="Sinfonia Concertanti">Sinfonia Concertanti</a></i> that was held at <a href="/wiki/St._Mary%27s_Church_Hall" title="St. Mary&#39;s Church Hall">St. Mary's Church Hall</a> at 7:30 in the evening on Sunday <a href="/wiki/7_May" title="7 May">7 May</a> in a year prior to <a href="/wiki/1997" title="1997">1997</a>, which featured <a href="/wiki/Emanuel_Hurwitz" title="Emanuel Hurwitz">Emanuel Hurwitz</a> on <a href="/wiki/Violin" title="Violin">violin</a>, <a href="/wiki/Kay_Hurwitz" title="Kay Hurwitz">Kay Hurwitz</a> on <a href="/wiki/Viola" title="Viola">viola</a>, and the <a href="/wiki/Dollis_Chamber_Ensemble" title="Dollis Chamber Ensemble">Dollis Chamber Ensemble</a>.<sup id="cite_ref-hple_1-0" class="reference"><a href="#cite_note-hple-1">&#91;1&#93;</a></sup> <a href="/wiki/Hermione_Granger" title="Hermione Granger">Hermione Granger</a> had a poster for this concert stuck up on the bulletin board in <a href="/wiki/Hermione_Granger%27s_bedroom" title="Hermione Granger&#39;s bedroom">her bedroom</a> in the <a href="/wiki/Granger_family" title="Granger family">Granger family</a>'s <a href="/wiki/Grangers%27_house" class="mw-redirect" title="Grangers&#39; house">house</a>.<sup id="cite_ref-hple_1-1" class="reference"><a href="#cite_note-hple-1">&#91;1&#93;</a></sup><sup id="cite_ref-dhp1_2-0" class="reference"><a href="#cite_note-dhp1-2">&#91;2&#93;</a></sup>
