<b>These Ministry Guards</b>&#160;were <a href="/wiki/Great_Britain" title="Great Britain">British</a> <a href="/wiki/Wizardkind" title="Wizardkind">wizards</a>&#160;who guarded the <a href="/wiki/Muggle-Born_Registration_Commission" title="Muggle-Born Registration Commission">Muggle-Born Registration Commission</a> alongside <a href="/wiki/Corban_Yaxley" title="Corban Yaxley">Corban Yaxley</a> and others in <a href="/wiki/1997" title="1997">1997</a>.
</p>
<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/wiki/Two_Unidentified_Ministry_Guards?veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Edit section: History">edit</a><span class="mw-editsection-divider"> | </span><a href="/wiki/Two_Unidentified_Ministry_Guards?action=edit&amp;section=1" title="Edit section: History"><svg class="wds-icon wds-icon-tiny section-edit-pencil-icon"><use xlink:href="#wds-icons-pencil-tiny"></use></svg>edit source</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In <a href="/wiki/1997" title="1997">1997</a>, these two Ministry Guard were stationed to guard the <a href="/wiki/Muggle-Born_Registration_Commission_courtroom" title="Muggle-Born Registration Commission courtroom">Muggle-Born Registration Commission courtroom</a> during the trial of <a href="/wiki/Mary_Cattermole" title="Mary Cattermole">Mary Cattermole</a>, overlooked by both <a href="/wiki/Dolores_Umbridge" title="Dolores Umbridge">Dolores Umbridge</a> and <a href="/wiki/Corban_Yaxley" title="Corban Yaxley">Corban Yaxley</a>. However, <a href="/wiki/Harry_Potter" title="Harry Potter">Harry Potter</a>, <a href="/wiki/Ronald_Weasley" title="Ronald Weasley">Ronald Weasley</a> and <a href="/wiki/Hermione_Granger" title="Hermione Granger">Hermione Granger</a>, both disguised using <a href="/wiki/Polyjuice_Potion" title="Polyjuice Potion">Polyjuice Potion</a> as <a href="/wiki/Albert_Runcorn" title="Albert Runcorn">Albert Runcorn</a>, <a href="/wiki/Reginald_Cattermole" title="Reginald Cattermole">Reginald Cattermole</a> and <a href="/wiki/Mafalda_Hopkirk" title="Mafalda Hopkirk">Mafalda Hopkirk</a> respectively, hijacked and stopped the corrupt trial, and <a href="/wiki/Stunning_Spell" title="Stunning Spell">stunned</a> these guards, along with Umbridge and Yaxley with offensive <a href="/wiki/Duelling" title="Duelling">duelling</a> <a href="/wiki/Spell" title="Spell">spells</a>, and rescued Mary Cattermole from being given a false and wrongful sentence by Umbridge.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup><sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup>
</p>
