This <b>individual</b> was a <a href="/wiki/Wizardkind" title="Wizardkind">wizard</a> who played as a <a href="/wiki/Beater" title="Beater">Beater</a> for the <a href="/wiki/Slytherin_Quidditch_team" title="Slytherin Quidditch team">Slytherin team</a> in <a href="/wiki/Quidditch" title="Quidditch">Quidditch</a>. In the <a href="/wiki/Slytherin" title="Slytherin">Slytherin</a> versus <a href="/wiki/Gryffindor_Quidditch_team" title="Gryffindor Quidditch team">Gryffindor</a> Hogwart's <a href="/wiki/Inter-House_Quidditch_Cup" title="Inter-House Quidditch Cup">Quidditch Cup</a> match in <a href="/wiki/1991" title="1991">1991</a>, he is shown to have his <a href="/wiki/Beater%27s_bat" title="Beater&#39;s bat">bat</a> taken by <a href="/wiki/Marcus_Flint" title="Marcus Flint">Marcus Flint</a>.<sup id="cite_ref-PSF_2-9" class="reference"><a href="#cite_note-PSF-2">&#91;2&#93;</a></sup>
</p>
