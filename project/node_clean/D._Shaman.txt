<b>Dr. D. Shaman</b> was a <a href="/wiki/Wizardkind" title="Wizardkind">wizard</a><sup id="cite_ref-gender_1-1" class="reference"><a href="#cite_note-gender-1">&#91;1&#93;</a></sup> doctor and advice columnist.<sup id="cite_ref-ootp-f_2-0" class="reference"><a href="#cite_note-ootp-f-2">&#91;2&#93;</a></sup><sup id="cite_ref-hbp-f_3-0" class="reference"><a href="#cite_note-hbp-f-3">&#91;3&#93;</a></sup>
</p><p>He wrote the "<a href="/wiki/Ask_D._Shaman" title="Ask D. Shaman">Ask D. Shaman</a>" advice column for the <i><a href="/wiki/Daily_Prophet" title="Daily Prophet">Daily Prophet</a></i> starting from the <a href="/wiki/29_November" title="29 November">November 29</a> <a href="/wiki/1926" title="1926">1926</a><sup id="cite_ref-4" class="reference"><a href="#cite_note-4">&#91;4&#93;</a></sup> issue onward.<sup id="cite_ref-ootp-f_2-1" class="reference"><a href="#cite_note-ootp-f-2">&#91;2&#93;</a></sup><sup id="cite_ref-5" class="reference"><a href="#cite_note-5">&#91;5&#93;</a></sup>
</p><p>He also wrote an article about <a href="/wiki/Wrackspurt" title="Wrackspurt">Wrackspurts</a>, entitled "Wrackspurts: Unfuss the Mystery", for a <a href="/wiki/1996" title="1996">1996</a> issue of <i><a href="/wiki/The_Quibbler" title="The Quibbler">The Quibbler</a></i>.<sup id="cite_ref-hbp-f_3-1" class="reference"><a href="#cite_note-hbp-f-3">&#91;3&#93;</a></sup>
</p>
